# ✂️ Dev Barber

Uma landing page moderna para barbearias, desenvolvida com HTML, Tailwind CSS e JavaScript. O projeto foi criado com foco em design responsivo, experiência do usuário e apresentação profissional dos serviços oferecidos.

## 🚀 Demonstração

🔗 Acesse o projeto online:

https://dev-barber-sable.vercel.app

## ✨ Funcionalidades

* Apresentação institucional da barbearia
* Exibição dos serviços oferecidos
* Horário de funcionamento
* Localização da empresa
* Botão de contato rápido
* Design responsivo para dispositivos móveis
* Animações para melhorar a experiência do usuário
